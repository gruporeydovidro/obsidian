## Tabelas

Você pode criar tabelas usando barras verticais (|) e hífens (-). As barras verticais separam as colunas e os hífens definem o cabeçalho da coluna.

  
~~~

| Primeiro nome | Sobrenome |
| ---------- | ---------- |
| máx | Plano |
| Maria | curie |

~~~

| Primeiro nome | Sobrenome |
| ------------- | --------- |
| máx           | Plano     |
| Maria         | curie     |



Resultado:

![](https://api-club-file.cb.hotmart.com/public/v5/files/169a3434-083e-41be-8a8a-8355cc79a420)

As barras verticais ou qualquer um dos lados da mesa são opcionais.

As células não precisam estar perfeitamente alinhadas com as colunas.   
Cada linha de cabeçalho deve ter pelo menos dois hífens.

| Primeiro nome | Sobrenome |
| : --- :       | ------- : |
|    teste      |   teste 2 |
| teste 3       |           |




---

### Alinhamento:

Você pode alinhar o texto à esquerda, à direita ou ao centro de uma coluna adicionando dois pontos (:) à linha do cabeçalho.

  

Texto alinhado à esquerda | Texto alinhado ao centro | Texto alinhado à direita

:-- | :--: | --:

Conteúdo | Conteúdo | Contente

 : teste | : teste2 : | teste3 :
 
 
 
|  Primeiro nome | Sobrenome |
| -------------- | --------- |
| :máx:          | Plano:     |
| :Maria:        | curie:     |

## Lista de Tarefas

As listas de tarefas são úteis quando você deseja criar uma lista de itens que podem ser marcados como concluídos ou não. Veja um exemplo:

~~~

- [x] Tarefa 1

- [ ] Tarefa 2

- [ ] Tarefa 3
      
~~~
- [x] Tarefa 1

- [ ] Tarefa 2

- [ ] Tarefa 3


Nesse caso, a primeira tarefa está marcada como concluída (o "x" entre colchetes), enquanto as outras duas ainda não foram concluídas.

### Código embutido

Você pode formatar o código dentro de uma frase usando acentos graves simples.

  

> O texto dentro de `backticks` em uma linha será formatado como código.

  
O texto dentro de backticks em uma linha será formatado como código.

### Blocos de código

Para formatar um bloco de código, coloque o código entre crases triplos. Pode-se us

  

``` js

cd ~/área de trabalho
teste
codigo

```

Você pode adicionar realce de sintaxe a um bloco de código, adicionando um código de idioma após o primeiro conjunto de acentos graves.
