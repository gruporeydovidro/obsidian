**Dica Importante: não se preocupe com estética.** O gráfico naturalmente vai ficando bonito, mas **não tente desfazer conexões entre as notas só porque elas não ficam bonitas no gráfico.** Deixe o conhecimento crescer de forma orgânica.

  

#### Entendendo o recurso de gráfico

O recurso de gráfico é uma das partes mais interessantes do Obsidian. Ele permite que você visualize todas as suas notas e suas conexões em um gráfico interativo. Cada nota é um ponto (ou nó) e cada link entre as notas é uma linha (ou aresta). Isso cria uma rede de conhecimento visual que você pode explorar e manipular.

  

#### Como o recurso de gráfico funciona

O recurso de gráfico usa um algoritmo de força para dispor as notas no espaço. Basicamente, cada nota é tratada como uma partícula carregada que repele todas as outras, enquanto cada link é tratado como uma mola que puxa suas notas conectadas juntas. O resultado é um layout que reflete a estrutura subjacente de suas notas e seus links.

  

---

#### Opções no Modo Gráfico:

Aqui estão as explicações das opções disponíveis na visualização de gráfico do Obsidian:

  

##### 1. Filtros:  

Os filtros permitem que você controle quais notas são mostradas no gráfico. Por exemplo, você pode querer ver apenas as notas que contêm uma determinada palavra-chave, ou apenas as notas que estão ligadas a uma nota específica.

Para criar um filtro, você precisa abrir o painel de filtros, que está localizado no canto inferior direito do gráfico. Lá, você pode inserir uma expressão de filtro. As expressões de filtro podem ser bastante complexas, mas geralmente você só precisará de algumas funções básicas. Por exemplo, `tag:#important` mostrará apenas as notas que têm a tag #important.

- **Search files (Buscar arquivos)**: permite que você filtre as notas com base em uma consulta de busca.  
- **Tags**: alterna a exibição de tags no gráfico.  
- **Attachments (Anexos)**: alterna a exibição de anexos no gráfico.  
- **Existing files only (Apenas arquivos existentes)**: alterna a exibição de notas que existem em seu cofre. Como uma nota não precisa existir para ser vinculada, isso pode ajudar a limitar seu gráfico às notas que você realmente tem em seu cofre.  
- **Orphans (Órfãos)**: alterna a exibição de notas sem quaisquer links.

  

##### 2. Grupos:  

Esta seção permite criar grupos de notas para distingui-las entre si usando cores. Para criar um novo grupo:

- Clique em **New group (Novo grupo)**.  
- Na caixa de pesquisa, digite uma consulta de pesquisa para as notas que você deseja adicionar ao grupo.  
- Clique no círculo colorido para dar uma cor ao grupo【29†source】.

  

##### 3. Tela (Display):  

Esta seção controla como visualizar nós e links no gráfico.

- **Arrows (Setas)**: alterna a exibição da direção de cada link.  
- **Text fade threshold (Limiar de desbotamento do texto)**: controla a transparência do texto para o nome de cada nota.  
- **Node size (Tamanho do nó)**: controla o tamanho do círculo que representa cada nota.  
- **Link thickness (Espessura do link)**: controla a largura da linha para cada link.  
- **Animate (Animar)**: inicia uma animação em lapsos de tempo【30†source】.

  

##### 4. Força:

No modo gráfico do Obsidian, existem quatro forças principais que controlam a aparência e a interação entre os nós (representando suas notas):

1. **Força do Centro (Center force)**: Esta força controla o quão compacto é o gráfico. Um valor mais alto cria um gráfico mais circular, trazendo os nós para mais perto do centro do gráfico.

2. **Força de Repulsão (Repel force)**: Esta força controla o quanto um nó empurra outros nós para longe dele. Ou seja, quanto maior o valor dessa força, mais espaço haverá entre os nós no gráfico.

3. **Força de Ligação (Link force)**: Esta força controla a atração de cada ligação. Se você pensar na ligação como uma banda de borracha, a força de ligação controla o quão apertada ou solta é a banda. Isso pode afetar o quão esticadas ou soltas as ligações aparecem no gráfico.

4. **Distância da Ligação (Link distance)**: Esta força controla o comprimento das linhas entre cada nota. Ou seja, ela determina quão longe os nós ligados ficarão um do outro no gráfico.

#### Melhores práticas

Aqui estão algumas dicas para aproveitar ao máximo o recurso de gráfico:

1. **Use links com moderação**: Não é preciso ligar todas as suas notas a todas as outras. Isso só tornaria o gráfico confuso e difícil de navegar. Em vez disso, tente criar links significativos que realmente ajudem a estruturar seu conhecimento.

2. **Use tags e filtros**: Tags e filtros são ferramentas poderosas para organizar suas notas e tornar o gráfico mais gerenciável. Use-os para destacar as notas mais importantes e ocultar as menos relevantes.

3. **Explore**: O gráfico é uma ferramenta exploratória. Use-o para descobrir conexões inesperadas entre suas notas e para gerar novas ideias.
