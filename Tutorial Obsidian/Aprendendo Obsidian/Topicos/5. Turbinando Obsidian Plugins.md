#### Hora de turbinar seu Obsidian

Os plugins são a cereja do bolo, são eles que podem tornar seu Obsidian muito além de um segundo cérebro, mas um segundo cérebro turbinado novas funcionalidades e potencializado com inteligência artificial.  
  

#### Plugins Nativos

São plugins que vêm com o próprio Obsidian. Eles são criados e mantidos pelos desenvolvedores do Obsidian, então você pode esperar um alto nível de qualidade e integração com o resto do aplicativo. Alguns exemplos de plugins nativos incluem o plugin de gráfico de conhecimento, que permite visualizar suas notas como um gráfico de rede, e o plugin de revisão diária, que ajuda a manter o controle de suas tarefas e eventos diários.  
  

#### Plugins Não Oficiais (ou melhor: da Comunidade Obsidian)

São plugins criados por membros da comunidade do Obsidian. Eles podem adicionar muitos recursos diferentes que vão além do que os plugins nativos oferecem. No entanto, como não são criados pelos desenvolvedores do Obsidian, eles podem variar em termos de qualidade e estabilidade, e podem não estar tão bem integrados com o resto do aplicativo.  
  

#### Plugins que eu recomendo para quem está começando:

##### 1. Essenciais:

**Recent Files**: Este plugin permite que você mantenha o controle dos arquivos que você usou recentemente, facilitando o acesso rápido a eles. É como o histórico do seu navegador, mas para o Obsidian.

**Janitor**: Um verdadeiro assistente de limpeza digital, este plugin ajuda a organizar e limpar o seu vault do Obsidian, removendo notas vazias, arquivos não utilizados e muito mais.

**Custom Frames**: Este plugin permite que você incorpore recursos externos como iframes dentro do seu Obsidian. Com ele, você pode adicionar a comunidade Obsidian e o ChatGPT diretamente na sua interface de usuário do Obsidian.

**Paste URL into selection**: Este plugin é um verdadeiro economizador de tempo. Ele permite que você transforme o texto selecionado em um link, simplesmente colando uma URL.

**Settings Search**: Este plugin é como um mecanismo de busca para as suas configurações do Obsidian. Ele permite pesquisar configurações e plugins na aba de configurações, ajudando você a encontrar rapidamente o que precisa.

**Language Translator**: Este plugin permite que você traduza qualquer texto dentro do Obsidian usando a Inteligência Artificial. É como ter um tradutor pessoal sempre à mão.

**Reading Time**: Este plugin oferece uma estimativa de quanto tempo levaria para ler em voz alta o texto que você escreveu. É uma ótima ferramenta para planejar a duração dos seus episódios de podcast ou vídeos.

**Better Word Count**: Este plugin oferece uma contagem detalhada de palavras e caracteres para o texto selecionado, útil para manter-se dentro dos limites de caracteres de outros plugins como o Language Translator e o ChatGPT.  
  

##### 2. Inteligência Artifical:

Ambos precisam de API da Open AI. (**»**[**clique aqui**](https://platform.openai.com/account/api-keys)**«**)

**Text Generator**: Este plugin utiliza a Inteligência Artificial para gerar texto. É como ter um escritor assistente ao seu lado, pronto para ajudar quando você estiver com dificuldades para encontrar as palavras certas.

**Smart Connection**: Este plugin utiliza a Inteligência Artificial para ajudar a criar conexões entre as suas notas. Ele sugere links relevantes entre as suas notas, ajudando a construir uma rede de conhecimento mais rica e interconectada em seu vault do Obsidian.  
  

##### 3. Super Plugins:

**Templater:** Turbinar o poder dos templates no seu Obsidian.

**Dateview:** Permite você tornar o seu Obsidian em um poderoso banco de dados.  

  

##### 4. Estou testando:

**Periodic Notes:** Este plugin é muito bom para journaling (diário), pois fornece um sistema para revisar notas diariamente, semanalmente, mensalmente, trimestralmente e anualmente.

**Excalidraw**: Este plugin permite que você crie desenhos e diagramas em suas notas. Não se preocupe se você não é um artista - é muito fácil de usar. É como ter um quadro branco digital em suas notas.

**CoPilot:** é um plugin para Obsidian que permite a integração com o ChatGPT. Parece ser melhor que Smart Connection, mas ainda está bem fase beta.

**MindMap:** Ainda testando, mas a princípio eu conseguiria substituir o MeisterMind com ele.

**Observação:** Lembre-se, a instalação e o uso de plugins de terceiros deve ser feita com cautela, pois eles não são oficialmente suportados pelos desenvolvedores do Obsidian. Sempre leia as instruções e as revisões antes de instalar um novo plugin. Uma dica é instalar os plugins que você quer testar, principalmente os novos e com poucos downloads, em um cofre separado do seu principal.
