#### Links Internos  

É aqui que a mágica começa a acontecer!

Isso é o que tornará o seu "segundo cérebro" algo incrível, provavelmente diferente de tudo o que você já tentou antes. Então, preste muita atenção, porque o verdadeiro poder do Obsidian começa com o sistema de ligação de notas.
```
Linkar é extremamente simples: basta selecionar o texto e clicar duas vezes em colchetes "[[". O texto selecionado será automaticamente linkado e, caso não exista uma nota com esse nome, ao clicar nele, você cria uma nova nota.
```

```
Se quiser linkar uma nota existente, basta digitar novamente os colchetes "[[" e começar a digitar o que está procurando. O Obsidian buscará a nota com o nome que você está digitando e mostrará todas as notas existentes.

```

```


Caso deseje linkar uma nota existente, mas com um nome diferente, você pode adicionar uma barra vertical "|" assim:

[[Nome da Nota Original| nome personalizado]]

A mecânica, digamos, de linkar notas é extremamente simples. O segredo está em saber o que linkar.

E isso você aprenderá no módulo Anotações Lendárias.
```

### Incorporar uma Nota dentro de outra Nota:

### Incorporando a partir de um Título…

```
Para inserir uma nota dentro de outra, basta fazer o mesmo processo de linkagem interna normal, ou seja, [[ mas com um ponto de exclamação antes:

![[[Nome da Nota]]  
```

Pronto, sua nota foi incorporada (embedada), ela agora está dentro de outra.

Aconselho fazer isso para notas pequenas, onde ao invés de repetir algo que você já escreveu, você simplesmente embeda para ficar mais fácil da leitura.

Contudo, se você tiver uma nota grande e quer só embedar uma parte dela, a solução é essa aqui:
```
![[[Nome da Nota#Titulo]]  
```

Você pode embedar a partir de títulos ou blocos.

Basta digitar:
```

![[Nome da Nota#  
```

E o próprio Obsidian irá te sugerir algumas opções para você embedar, incorporar, um pedaço da sua nota.

**Exemplo de Título:**

**Quando digito o nome da nota e coloco # aparece esta opção:**

![](https://api-club-file.cb.hotmart.com/public/v5/files/0617b6c4-e3b0-412f-8746-29e1a0738cd7)

**E quando clico no título que eu quero acontece isso:**

![](https://api-club-file.cb.hotmart.com/public/v5/files/04a1eb58-249c-47a2-8120-6368fb177639)

Neste exemplo eu embedei apenas uma parte dessa nota, é uma nota com mais de 6000 palavras, foi onde escrevi o episódio 021 do Vida Lendária.

Ele trouxe apenas o que estava dentro dessa headline, ou seja, quando o outro título começa, ele interrompe o texto que foi incorporado.

Assim, o que eu alterar lá, altera diretamente aqui, sem precisar ter conteúdo duplicado.  
Isso é EXTREMAMENTE útil se for bem utilizado.

### Incorporando a partir de um Bloco…

Você pode também embedar a partir de um bloco de outra nota.

Basta digitar:
```

![[Nome da Nota#^]]  
```

E o próprio Obsidian irá te sugerir algumas opções para você embedar, incorporar, um pedaço da sua nota.

**Exemplo de Bloco:**
```

![[021 - Seja um Otimista Racional#^f08c08]]  
```

E no editor visual fica assim:

![](https://api-club-file.cb.hotmart.com/public/v5/files/9567a19c-1c3a-4bb9-ae6a-b73fd8bbad58)

**Arquivos e Imagens**

Para inserir arquivos como PDF, MP3 e imagens é extremamente simples, basta arrastar o arquivo e largar dentro da sua nota. O Obsidian automaticamente irá embedar, ou seja, permitir que você execute o áudio ou leia o PDF dentro do seu documento. 

Importante: Se você não definir uma pasta para ele, o Obsidian colocará todos arquivos na sua pasta principal, eu prefiro criar uma pasta chamada Anexos. Para alterar o local onde ele salva siga este caminho:

Configurações > Arquivos & Links > Local da pasta de anexos

Insira o nome da sua pasta, ela pode ter o nome que você achar que faz mais sentido, originalmente eu chamava de Imagens, mas depois comecei a colocar vários tipos diferentes de arquivos e hoje chamo de Anexos.

## Links Externos

### Linkar URL

Você pode apenas colar o link, ou se desejar, colar o link em cima de uma palavra e ela ficará automaticamente linkada.

Mas se quiser editar o markdown é bem simples, basta colocar o texto do link entre colchetes ('[ ]') e, em seguida, a URL entre parênteses ('( )').

Fica assim:
```

[Documentação Obsidian](https://help.obsidian.md)  
```

Bem simples, mas você não precisa fazer isso se não quiser, basta selecionar o texto e colar a URL por cima onde deseja e ele fará automatiacmente isso para você.

### Incorpore um vídeo do YouTube

Existe uma forma fácil e uma nerd de fazer isso.

**A forma fácil é:**
```

![]( aqui o link do vídeo do YouTube )
```

Exemplo:
```


![](https://www.youtube.com/watch?v=NnTvZWp5Q7o)  
```

Isso é o suficiente para ele ficar dentro já do seu Obsidian pronto para dar o play.

**Temos também a forma “nerd”:**

A forma nerd é fazer com iframe, já que o Obsidian lê HTML e assim você pode definir tamanho e outras configurações. Seria assim:
```


<iframe src="INSIRA SEU URL AQUI"></iframe>  
```

Mas pra que complicar né?

Bora fazer da forma fácil que é:
```
![](o que você deseja inserir)  
```

E serve não só para vídeos do YouTube, mas para Tweets, Imagens, GIFs e muitos outros.
