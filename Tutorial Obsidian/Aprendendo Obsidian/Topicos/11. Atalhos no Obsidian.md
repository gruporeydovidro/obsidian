#### ​- Mouse = + Produtividade  

Esses atalhos adicionais podem ajudar a tornar o uso do Obsidian ainda mais eficiente, permitindo que você execute ações comuns mais rapidamente e com menos interrupções no fluxo de trabalho.  
  

#### Atalhos

**Ctrl/Cmd + O**: Este atalho permite abrir rapidamente um arquivo. É importante porque permite que você acesse rapidamente suas notas sem ter que navegar através do painel de arquivos.

**Ctrl/Cmd + N**: Este atalho cria uma nova nota. É útil para capturar rapidamente novas ideias ou informações sem ter que navegar para a opção de criar uma nova nota.

**Ctrl/Cmd + F**: Este atalho abre a barra de pesquisa. É útil para encontrar rapidamente informações em suas notas.

**Ctrl/Cmd + P**: Este atalho abre a paleta de comandos. É útil para acessar rapidamente todos os comandos disponíveis no Obsidian.

**Ctrl/Cmd + T**: Abrir nova aba vazia. Clicar em uma nota com **Ctrl/Cmd** abre essa aba em uma nova guia.

**Ctrl/Cmd + Shift + V**: Este atalho cola o texto sem formatação. É útil para colar texto de outras fontes sem trazer a formatação original.

**Ctrl/Cmd + K**: Este atalho cria um link interno. É útil para conectar rapidamente suas notas e criar uma rede de conhecimento.

**Ctrl/Cmd + B**: Este atalho adiciona formatação em negrito ao texto selecionado. É útil para destacar informações importantes em suas notas.

**Ctrl/Cmd + Shift + F**: Este atalho abre a barra de pesquisa no arquivo atual. É útil para encontrar rapidamente informações na nota atual.

**Ctrl/Cmd + S**: Este atalho salva a nota atual. É útil para garantir que suas alterações sejam salvas rapidamente.

**Ctrl/Cmd + Shift + K**: Este atalho exclui a linha atual. É útil para remover rapidamente informações desnecessárias ou erros.

**Ctrl/Cmd + = e - | Ctrl/Cmd + 0** : Estes atalhos aumentam e diminuem o zoom na nota atual. São úteis para focar em detalhes específicos ou ter uma visão geral da nota.

**Ctrl/Cmd + E**: Este a`talho alterna entre o modo de edição e o modo de visualização. É útil para revisar rapidamente suas notas e fazer edições rápidas.
